# PulseOps AI

## AI Powered Production Observability Platform

PulseOps AI is a production-grade observability platform built to learn Linux, Docker, ClickHouse, Kafka, Redis, Nginx, Google Cloud and AI in a real-world architecture.

## Tech Stack

- Linux
- Docker
- Docker Compose
- ClickHouse
- Kafka
- Redis
- Node.js
- React
- Nginx
- Google Cloud
- GitHub Actions
- Gemini AI

## Goal

Build a production-ready monitoring platform capable of:

- Collecting application events
- Processing millions of logs
- Real-time analytics
- AI-generated incident summaries
- Deployment on Google Cloud
